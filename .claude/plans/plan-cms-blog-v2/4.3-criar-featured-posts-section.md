# ✅ 4.3 — Criar `featured-posts-section.tsx`

## Contexto
Seção que aparece no topo do `/blog` listando posts com
`isFeaturedGlobal: true` + status published. Server component que recebe
posts já fetchados pela página (não fetcha nem cacheia internamente — a
página em 4.16 que controla o cache).

## Detalhe técnico
Criar `apps/web/modules/blog/ui/components/featured-posts-section.tsx`:

Props:
```ts
interface FeaturedPostsSectionProps {
  posts: Array<{
    _id: string;
    slug: string;
    title: string;
    excerpt: string;
    coverImage: string;
    publishedAt?: string | Date | null;
    tools?: Array<{ _id: string; title: string }>;
  }>;
}
```

Comportamento:
- Se `posts.length === 0`, retornar `null` (não renderiza nada).
- Heading `<h2 className="text-2xl font-bold tracking-tight text-foreground">Em destaque</h2>`.
- Grid responsivo: `grid grid-cols-1 gap-6 md:grid-cols-2 lg:grid-cols-3`.
- Cada item renderizado via `<PostCard post={post} />` (importar de
  `./post-card`).

Estrutura do JSX:
```tsx
<section className="mb-12">
  <h2 className="mb-6 text-2xl font-bold tracking-tight text-foreground">Em destaque</h2>
  <div className="grid grid-cols-1 gap-6 md:grid-cols-2 lg:grid-cols-3">
    {posts.map((p) => <PostCard key={p._id} post={p} />)}
  </div>
</section>
```
