# ✅ 3.2b — Criar `prefetch` server-side dedicado

## Contexto
Hidratação SSR em `/dashboard/posts` exige prefetch das queries tRPC. Análogo dedicado de `apps/web/modules/common/prefetch.ts` (regra de não-reuso).

## Detalhe técnico
Criar `apps/web/modules/dashboard/posts/server/prefetch.ts`:

```ts
import { prefetch, trpc } from "@/trpc/server";
import type { inferInput } from "@trpc/tanstack-react-query";

type ListInput = inferInput<typeof trpc.posts.list>;

export const prefetchPosts = (params: ListInput) => {
  return prefetch(trpc.posts.list.queryOptions(params));
};

export const prefetchPost = (id: string) => {
  return prefetch(trpc.posts.getById.queryOptions({ id }));
};
```

NÃO tocar `apps/web/modules/common/prefetch.ts`.
