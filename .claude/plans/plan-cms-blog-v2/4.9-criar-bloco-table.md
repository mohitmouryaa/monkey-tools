# ✅ 4.9 — Criar bloco renderer: Table

## Contexto
Renderiza um bloco `table` salvo pelo `@editorjs/table`. Server component
(P2 = A).

## Detalhe técnico
Criar `apps/web/modules/blog/ui/components/blocks/table-block.tsx`.

Shape:
```json
{
  "type": "table",
  "data": {
    "withHeadings": true,
    "content": [
      ["Coluna A", "Coluna B"],
      ["valor 1", "valor 2"]
    ]
  }
}
```

```tsx
interface TableBlockData {
  withHeadings?: boolean;
  content?: string[][];
}

export const TableBlock = ({ data }: { data: TableBlockData }) => {
  const rows = data.content ?? [];
  if (rows.length === 0) return null;

  const [headRow, ...bodyRows] = data.withHeadings ? rows : [null, ...rows];

  return (
    <div className="overflow-x-auto my-6">
      <table className="w-full">
        {headRow && (
          <thead>
            <tr>
              {headRow.map((cell, i) => (
                <th
                  key={i}
                  // biome-ignore lint/security/noDangerouslySetInnerHtml: cells contém HTML inline do Editor.js
                  dangerouslySetInnerHTML={{ __html: cell }}
                />
              ))}
            </tr>
          </thead>
        )}
        <tbody>
          {bodyRows.map((row, ri) => (
            <tr key={ri}>
              {row.map((cell, ci) => (
                <td
                  key={ci}
                  // biome-ignore lint/security/noDangerouslySetInnerHtml: idem
                  dangerouslySetInnerHTML={{ __html: cell }}
                />
              ))}
            </tr>
          ))}
        </tbody>
      </table>
    </div>
  );
};
```

Notas:
- `prose` no wrapper externo dá estilo a `<table>`/`<th>`/`<td>`.
- `overflow-x-auto` previne quebra de layout em tabelas largas no mobile.
