# ✅ 4.7 — Criar bloco renderer: Quote

## Contexto
Renderiza um bloco `quote` salvo pelo `@editorjs/quote`. Server component
(P2 = A).

## Detalhe técnico
Criar `apps/web/modules/blog/ui/components/blocks/quote-block.tsx`.

Shape:
```json
{
  "type": "quote",
  "data": {
    "text": "Não é o que você diz, é o que você faz.",
    "caption": "Anônimo",
    "alignment": "left"
  }
}
```

```tsx
interface QuoteBlockData {
  text?: string;
  caption?: string;
  alignment?: "left" | "center";
}

export const QuoteBlock = ({ data }: { data: QuoteBlockData }) => {
  const text = data.text ?? "";
  const caption = data.caption ?? "";
  return (
    <blockquote>
      {/* biome-ignore lint/security/noDangerouslySetInnerHtml: Editor.js inline tools sanitiza no admin */}
      <p dangerouslySetInnerHTML={{ __html: text }} />
      {caption && (
        <cite
          // biome-ignore lint/security/noDangerouslySetInnerHtml: idem
          dangerouslySetInnerHTML={{ __html: caption }}
        />
      )}
    </blockquote>
  );
};
```

Notas:
- Não tratar `alignment` no v1 — `prose` aplica estilo padrão de
  blockquote. Adicionar só se design pedir.
