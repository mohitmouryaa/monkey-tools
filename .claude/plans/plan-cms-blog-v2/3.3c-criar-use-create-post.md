# 3.3c — Criar hook `use-create-post` ✅ 

## Contexto
Mutation para `posts.create` com toast + invalidação de `posts.list`.

## Detalhe técnico
Criar `apps/web/modules/dashboard/hooks/use-create-post.tsx`:

```tsx
"use client";
import { toast } from "sonner";
import { useTRPC } from "@/trpc/client";
import { useMutation, useQueryClient } from "@tanstack/react-query";

export const useCreatePost = () => {
  const trpc = useTRPC();
  const queryClient = useQueryClient();

  return useMutation(
    trpc.posts.create.mutationOptions({
      onSuccess: (data) => {
        toast.success(`Post "${data.title}" created`);
        queryClient.invalidateQueries({ queryKey: trpc.posts.list.queryKey() });
      },
      onError: (error) => {
        toast.error(`Failed to create post: ${error.message}`);
      },
    }),
  );
};
```

Padrão alinhado com `use-create-tool.tsx` (toast + invalidate). Usar `queryKey()` para invalidar todas as variantes paginadas.
