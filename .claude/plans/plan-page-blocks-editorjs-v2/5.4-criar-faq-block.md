# ✅ 5.4 — Criar `<FaqBlock>`

## Contexto

Acordeao de perguntas e respostas usando `<Accordion>` do `@workspace/ui`. As respostas chegam como string HTML (vinda do editor — Editor.js inline tools geram HTML basico tipo `<a>`, `<strong>`, `<em>`). Renderizamos via `dangerouslySetInnerHTML` com classes utilitarias arbitrarias.

## Detalhe tecnico

Criar arquivo novo. **NAO precisa `"use client"` no topo do arquivo** — o `<Accordion>` shadcn ja eh client (`packages/ui/src/components/accordion.tsx:1` tem `"use client"`). Server component pode importar e usar normalmente; React server components compoem fronteiras automaticamente.

**Imports:**
- `import { Accordion, AccordionItem, AccordionTrigger, AccordionContent } from "@workspace/ui/components/accordion";`
- `import type { FaqBlockData } from "@workspace/types";`

**Props:**
```ts
interface FaqBlockProps {
  data: FaqBlockData;
}
```

**Estrutura visual:**

```tsx
const items = data.items ?? [];

return (
  <section className="py-16 md:py-20">
    <div className="max-w-3xl mx-auto px-4 sm:px-6">
      {data.title && (
        <h2 className="text-3xl md:text-4xl font-bold text-foreground mb-8 md:mb-10 text-center">
          {data.title}
        </h2>
      )}

      <Accordion type="single" collapsible className="w-full">
        {items.map((item, index) => (
          <AccordionItem key={`faq-${index}`} value={`faq-${index}`}>
            <AccordionTrigger className="text-base md:text-lg font-semibold">
              {item.question}
            </AccordionTrigger>
            <AccordionContent>
              <div
                className="text-sm md:text-base text-muted-foreground leading-relaxed [&_a]:text-primary [&_a]:underline [&_a]:underline-offset-[3px] [&_strong]:font-semibold [&_strong]:text-foreground [&_em]:italic"
                // biome-ignore lint/security/noDangerouslySetInnerHtml: editor inline tools sanitiza no admin
                dangerouslySetInnerHTML={{ __html: item.answer }}
              />
            </AccordionContent>
          </AccordionItem>
        ))}
      </Accordion>
    </div>
  </section>
);
```

**Notas:**
- `type="single" collapsible` — apenas um item aberto por vez, podendo fechar tudo. Padrao mais comum em FAQ institucional.
- Comentario `biome-ignore lint/security/noDangerouslySetInnerHtml` segue o mesmo precedente do `header-block.tsx:32`.
- Container `max-w-3xl` — FAQ tradicionalmente eh coluna unica de leitura confortavel.
- Sem subtitulo aqui — `FaqBlockData` so tem `title`.
