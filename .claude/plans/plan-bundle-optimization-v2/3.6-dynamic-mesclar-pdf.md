# ✅ 3.6 — Tornar dynamic `pdf-lib` em `mesclar-pdf.tsx`

## Contexto

Tool usa `pdf-lib` no handler de merge (em torno da linha 95). Linha 6:
`import { degrees, PDFDocument } from "pdf-lib";`. `degrees` é importado mas
talvez não usado neste arquivo — verificar e remover se não estiver em uso
real (search `degrees(` no arquivo).

## Detalhe técnico

Em `apps/web/modules/tools/ui/components/mesclar-pdf.tsx`:

1. **Remover linha 6**:
   ```ts
   import { degrees, PDFDocument } from "pdf-lib";
   ```

2. **Adicionar import do util**:
   ```ts
   import { lazyLoadPdfLib } from "@/modules/common/lib/lazy-load-libs";
   ```

3. **No handler de merge** (procurar `mergedPdfDoc = await PDFDocument.create()` em torno da linha 95), antes do primeiro uso de `PDFDocument.create()`, adicionar:
   ```ts
   const { PDFDocument, degrees } = await lazyLoadPdfLib();
   ```
   Se `degrees` não for usado no arquivo (verificar via `grep -n "degrees(" mesclar-pdf.tsx`), destrucurar apenas `PDFDocument`.

4. Demais usos continuam funcionando.
