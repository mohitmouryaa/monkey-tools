# ✅ 3.9 — Tornar dynamic `jspdf` em `jpg-para-pdf.tsx`

## Contexto

Tool importa `jsPDF` no topo (linha 3: `import jsPDF from "jspdf";`, default
export, ~407 KB) e usa só em `convertToPdf` (em torno da linha 106:
`const pdf = new jsPDF();`).

## Detalhe técnico

Em `apps/web/modules/tools/ui/components/jpg-para-pdf.tsx`:

1. **Remover linha 3**:
   ```ts
   import jsPDF from "jspdf";
   ```

2. **Adicionar import do util**:
   ```ts
   import { lazyLoadJsPdf } from "@/modules/common/lib/lazy-load-libs";
   ```

3. **No handler `convertToPdf`** (em torno da linha 106), antes de `new jsPDF()`, adicionar:
   ```ts
   const { default: jsPDF } = await lazyLoadJsPdf();
   const pdf = new jsPDF();
   ```
   (Substituindo a linha existente `const pdf = new jsPDF();` por essas duas.)

## Arquivos tocados

- `apps/web/modules/tools/ui/components/jpg-para-pdf.tsx` — remove import top-level de jspdf, dynamic load no handler
